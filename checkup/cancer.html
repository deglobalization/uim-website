공단5대암검진

고운속내과의원 공단5대암검진 안내입니다.

공단5대암검진이란?

국민건강보험공단에서 제공하는 5대 암(위암, 대장암, 간암, 유방암, 자궁경부암) 검진 프로그램으로, 암을 조기에 발견하여 치료 성공률을 높이고 사망률을 낮추는 것을 목적으로 합니다.

검진 대상

  • 위암: 만 40세 이상, 2년마다 1회
  • 대장암: 만 50세 이상, 1년마다 1회
  • 간암: 만 40세 이상 고위험군, 6개월마다 1회
  • 유방암: 만 40세 이상 여성, 2년마다 1회
  • 자궁경부암: 만 20세 이상 여성, 2년마다 1회

검진 비용

공단5대암검진은 국민건강보험공단에서 비용의 일부를 부담하며, 수검자는 본인부담금(10%)만 납부하면 됩니다.

의료급여수급권자 및 건강보험료 하위 50%는 본인부담금 없이 무료로 검진을 받을 수 있습니다.

검진 항목

위암 검진

위암은 우리나라에서 발생률이 높은 암으로, 조기 발견 시 완치율이 높습니다.

검사 방법

  • 위내시경검사: 내시경을 통해 위 내부를 직접 관찰하는 검사
  • 위장조영검사(선택): 조영제를 마신 후 X-ray로 위를 관찰하는 검사

준비 사항

  • 검사 전 8시간 이상 금식 (물 포함)
  • 위내시경 검사 시 수면/비수면 선택 가능
  • 수면 내시경 선택 시 보호자 동반 필요

대장암 검진

대장암은 조기에 발견하면 90% 이상의 완치율을 보이는 암입니다.

검사 방법

  • 분변잠혈검사(FOBT): 대변에 혈액이 섞여 있는지 확인하는 검사
  • 대장내시경검사: 분변잠혈검사 양성 시 시행하는 정밀검사

준비 사항

  • 분변잠혈검사: 특별한 준비 필요 없음
  • 대장내시경: 검사 전날부터 특별한 장 세척 필요
  • 대장내시경은 수면으로 진행

간암 검진

간암은 B형 간염, C형 간염, 간경변 등의 고위험군에서 주로 발생합니다.

검사 방법

  • 간초음파검사: 초음파를 이용해 간의 종양 유무를 확인
  • 혈청알파태아단백검사(AFP): 간암 관련 종양표지자 검사

검진 대상

  • 만 40세 이상의 B형 간염 바이러스 항원 양성자
  • 만 40세 이상의 C형 간염 바이러스 항체 양성자
  • 만 40세 이상의 간경변증 환자

유방암 검진

유방암은 여성에게 발생하는 암 중 발생률이 높은 암입니다.

검사 방법

  • 유방촬영술(Mammography): X-ray를 이용한 유방 촬영
  • 유방초음파(선택): 필요 시 추가 검사

준비 사항

  • 검사 당일 유방 부위에 파우더, 로션 등 사용 금지
  • 생리 직전이나 생리 중인 경우 검사 연기 권장
  • 임신 중이거나 수유 중인 경우 사전 상담 필요

자궁경부암 검진

자궁경부암은 조기 발견 시 완치율이 매우 높은 암입니다.

검사 방법

  • 자궁경부세포검사(Pap smear): 자궁경부에서 세포를 채취하여 검사
  • 인유두종바이러스(HPV) 검사(선택): 필요 시 추가 검사

준비 사항

  • 생리 중인 경우 검사 연기 권장
  • 검사 2일 전부터 질 세정, 질 내 약물 사용 금지
  • 검사 2일 전부터 성관계 금지
